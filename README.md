# 🃏 Super Trunfo: Cadastro de Cartas de Cidades

Este projeto é um sistema de cadastro de cartas para um jogo de "Super Trunfo" baseado em estados e cidades brasileiras. Foi desenvolvido como parte dos meus estudos em **Linguagem C**, focando na manipulação de tipos primitivos e entrada/saída de dados.

## 🚀 Funcionalidades
- Cadastro de informações básicas de cidades (Estado, Código, Nome).
- Armazenamento de dados estatísticos (População, Área, PIB).
- Contagem de pontos turísticos para mecânicas de jogo.

## 🛠️ Tecnologias Utilizadas
- **Linguagem C**: Para a lógica de programação e manipulação de memória.
- **GCC**: Compilador utilizado para rodar o código.
- **Markdown**: Para a documentação do repositório.
